quantum field theory

Feynman Parameter

$
{\pi ^2}\int_0^1 \frac{ \rm dx }{ \sqrt {(a^2x + b^2)(1 - x)} } = \frac{ 2\pi ^2 }{a}arcsin\left( \frac{a}{ \sqrt { {a^2} + {b^2} } } \right)
$
证明：

首先利用：
对$k$的全空间积分，平移$\vec k$结果不变，积分一次得到：
$\int_{ - \infty }^\infty {\frac{ {\rm d ^3}k }{\left( {\vec k + \vec a} \right)}^2}(k^2 + b^2) = {\pi ^2}\int_0^1 \frac{\rm{dx}}{\sqrt \Delta } $
将$\Delta$显式写出$\Delta = (a^2x + b^2)(1 - x)$：
再积分一次得到：
$
{\pi ^2}\int_0^1 \frac{ \rm dx }{ \sqrt {(a^2x + b^2)(1 - x)} } = \frac{ 2\pi ^2 }{a}arcsin\left( \frac{a}{ \sqrt { {a^2} + {b^2} } } \right)
$
这样我们就证明了这个结果 Q.E.D.
