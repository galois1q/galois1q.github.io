从诺特定理得到对称性

作用量理论简介

这一节我们来回顾一下分析力学的基础概念.分析力学的整体框架是建立在哈密顿方程或者拉格朗日方程的体系上的,
而二者实际又相互联系.

对于一个粒子,我们利用广义坐标$q{_i}$及其一阶导数$\dot q{_i}$描述它某一时刻的位置和运动趋势（速度）,利用Lagrange量描述它在某一时刻的状态,当粒子随时间变化开始运动,Lagrange量也随之变化.而Lagrange量随时间的累计（即积分）相当于粒子运动的某种路径,反映了粒子随时间的演化情况,我们称它为作用量.
从状态a到b的作用量是:

$S=\int_{a}^{b}L(q_i,{\dot q_i},t)dt$

（其中指标i标记变量坐标分量）

在分析力学中有最小作用量原理要求
作用量是处处极小的.即

$\delta S \equiv 0$

利用变分法可导出熟知的Euler-Lagrange 方程:

$ \frac{ {\rm d} }{ { {\rm d}t } }( \frac{ { \partial L } }{ { \partial { \dot q _i} } } ) - \frac{ { \partial L } }{ { \partial {q_i} } } = 0 $

数"的概念描述泛函的变化,对于一个泛函F[f(x)]F[f(x)]F[f(x)],它的泛函导数定义如下：

$ \frac{{\delta {\rm{F}}}}{{\delta f(x)}} = \mathop {\lim }\limits_{^{\varepsilon \to 0}} \frac{{F[f(x') + \varepsilon \delta (x - x')] - F[f(x')]}}{\varepsilon } $

利用泛函导数将最小作用量写成如下形式：

$\frac{{\delta S}}{{\delta {q_i}(t)}} = 0$

利用泛函导数展开

$ \begin{array}{l} \frac{{\delta S}}{{\delta {q_i}(t)}} = \int_a^b {{\rm{du}}} [\frac{{\delta L}}{{\delta {q_i}(u)}}\frac{{\delta {q_i}(u)}}{{\delta {q_i}(t)}} + \frac{{\delta L}}{{\delta {{\dot q}_i}(u)}}\frac{{\delta {{\dot q}_i}(u)}}{{\delta {q_i}(t)}}]\\ = \int_a^b {{\rm{du}}} [\frac{{\delta L}}{{\delta {q_i}(u)}}\delta (u - t) + \frac{{\delta L}}{{\delta {{\dot q}_i}(u)}}\frac{{\rm{d}}}{{{\rm{dt}}}}\delta (u - t)]\\ = \frac{{\delta L}}{{\delta {q_i}(t)}} + [\delta (u - t)\frac{{\delta L}}{{\delta {{\dot q}_i}(u)}}]_a^b - \int_a^b {{\rm{du}}} \delta (u - t)\frac{{\rm{d}}}{{{\rm{dt}}}}\frac{{\delta L}}{{\delta {{\dot q}_i}(u)}}\\ = \frac{{\delta L}}{{\delta {q_i}(u)}} - \frac{{\rm{d}}}{{{\rm{dt}}}}\frac{{\delta L}}{{\delta {{\dot q}_i}(t)}} \end{array} $

就得到了Euler-Lagrange 方程

$ \frac{{\delta L}}{{\delta {q_i}(u)}} - \frac{{\rm{d}}}{{{\rm{dt}}}}\frac{{\delta L}}{{\delta {{\dot q}_i}(t)}} = 0 $

一般情况下,可将$\delta$算子换成偏微分算子$\rm \partial$（二者的计算一 样）,就是常见形式.

取Lagrange量为$L = \sum\limits_i {\frac{1}{2}m{{\dot q}_i}^2 - V({q_i})} $ 利用E-L方程,得到$m\ddot q = - \frac{{\partial V}}{{\partial q}}$
这正是牛顿第二定律

Euler-Lagrange方程也可以写成更加对称的形式,令

$ \begin{array}{l} \frac{{\partial L}}{{\partial {{\dot q}_i}}} = {p_i},\frac{{\partial L}}{{\partial {q_i}}} = {{\dot p}_i}\\ {\rm H} = \sum\limits_i {{p_i}} {{\dot q}_i} - L \end{array} $ 即有Hamilton正则方程,实际上Hamilton量通常就是所谓的体系能量: $ \frac{{\partial H}}{{\partial {q_i}}} = - {{\dot p}_i},\frac{{\partial {\rm H}}}{{\partial {p_i}}} = {{\dot q}_i} $ 而在力学中,我们称不随时间变化的量称为守恒量.守恒量是物理学家最喜欢的量, 因为当我们建立起体系的Lagrange方程或者Hamilton方程后,要了解体系只要求解这些方程就行. 而守恒量可以大大简化计算,如同求解一元代数方程知道几个根就可用韦达定理推算剩余的根,而非利用求根公式逐步化简. 而一个力学量A随时间的变化在Hamilton方程的帮助下可写成极其简洁的形式: $ \frac{{{\rm d}A}}{{{\rm d}t}} = \frac{{\partial A}}{{\partial t}} + \{ {\rm H},A \} $ $ \{ f,g\} = \sum\limits_i {\left( {\frac{{\partial f}}{{\partial {q_i}}}\frac{{\partial g}}{{\partial {p_i}}} - \frac{{\partial f}}{{\partial {p_i}}}\frac{{\partial g}}{{\partial {q_i}}}} \right)} $

其中

$ f = f\left( {{{\rm{q}}_i},{p_i}} \right),g = g\left( {{{\rm{q}}_i},{p_i}} \right) $

通常称${ f,g} $为f和g的泊松括号,如果泊松括号为零,我们称二者是对易的.

$\{ {{\rm{q}}_i},{p_j}\} = \delta _j^i,\{ {{\rm{q}}_i},{q_j}\} = \{ {{\rm{p}}_i},{p_j}\} = 0$

值得注意的是,这些关系通常只在完整约束下成立,对于一类不完整约束我们需要采用Dirac括号代替Poisson括号.可参看Weinberg《lectures on quantum mechanics》第九章

通俗地讲,以上公式只是在自由粒子的和一些类似于不需要讨论粒子是否会从曲面飞离或是不用考虑静不定的情形成立.之所以称之为完整约束,也是这个时候系统的自由度和广义坐标的自由度恰好相等,不会出现广义坐标作为变量冗余或是缺少.
这一点在二次量子化体现的更为深刻,当我们试图量子化电磁场时,电磁场的四维矢势和对应的正则动量是互相约束的,不能直接将Poisson括号变为对易子.

实际上,在一般的电动力学教材中,我们也是通过Larange量密度得到Maxwell方程,因为这是一个Lorentz不变标量,变分时未引入正则动量带来约束时更为容易处理,在得到Lagrange量之后再得到Hamilton量.那为何在二次量子化时不直接使用Lagrange量处理呢？
本人学识浅陋,认为原因在于难以有一般的简洁方法直接从Lagrange量来量子化.

  1. $\{ f,g\} = - \{ g,f\} $
  2. $\{\alpha f+\beta g ,h\}=\alpha\{f,h\}+\beta\{g,h\}$
  3. $\{f,\{g,h\}\}+\{g,\{h,f\}\}+\{h,\{g,f\}\}=0 $(Jacobi恒等式)

特别的:如果$A$不显含时间,即变量里面没有直接的含$t$项,则有

$ \{ {\rm H},A\} = 0 $

可以看出,一个变量不直接含时间的物理量守恒就是其和哈密顿量对易,后面将看到这种形式讨论对称性是方便的.

对称变换及Noether定理的导出

为了导出Noether’s theorem,我们将利用分析力学方法讨论连续对称性.我们将会看到对称性决定着守恒量.
实际在物理学中,这一规律是普适的.例如相位不变性导出电荷守恒,空间平移对称性导出动量守恒,甚至更广义的角度,对称性决定系统的一切.

那如何描绘用数学语言描绘对称性？所谓对称性,可以看作我们改变坐标参变量下系统整体不变,即描述系统的作用量在变换对应下具有形式不变性:

$ q(t)_i \to \widetilde q(t)_i,S[q(t)_i] \to S[\widetilde q(t)_i] $

如果我们可以推出此时Hamilton量是形式不变的,那么作用量自然是形式不变的.
考虑无穷小变换下有:

$ {{\rm{q}}_k} \to {{{\rm{q'}}}_k} = {{\rm{q}}_k} + \epsilon {f_k}\left( q \right) $

考虑变换的Jacobi行列式我们有:

$ {\Lambda _{ij}} = \frac{{\partial {{{\rm{q'}}}_i}}}{{\partial {{\rm{q}}_j}}} \simeq {\delta _{ij}} + \frac{{\epsilon \partial {f_i}\left( q \right)}}{{\partial {{\rm{q}}_j}}} $

由此我们可以写出动量的变换关系式,

$ {p_i} \to \sum\limits_j {{p_j}} {\Lambda _{ji}}^{ - 1} \simeq {p_i} - \sum\limits_j {{p_j}\frac{{\epsilon \partial {f_j}\left( q \right)}}{{\partial {{\rm{q}}_i}}}} $

定义$Q = \sum\limits_k {{p_k}{f_k}\left( q \right)}$, 在无穷小变换下哈密顿量形式不变,即 $ {\rm H}\left( {{{{\rm{q'}}}_k},{{p'}_k}} \right) - {\rm H}\left( {{{\rm{q}}_k},{p_k}} \right) = \epsilon \left\{ {\rm H} \right.,\left. Q \right\} = 0 $
由 $\{ {\rm H},Q\} = 0$则Q守恒,我们的确看到在对称性下始终存在着守恒量,这就是Noether’s theorem.
此外,如果Lagrange量相差一个关于时间的全导数$\frac{{{\rm d}G}}{{{\rm d}t}}$,拉格朗日量或哈密顿量不是形式不变的,
但对Lagrange量积分,容易发现此时作用量是不变的,那么这种情况下守恒量依然存在.事实上,推导Lagrange方程中求导后E-L方程是一样的；通过与前面相似的推导可以得到守恒量形式为:

$ Q = \sum\limits_k {{p_k}{f_k}\left( q \right)} + G $

值得指出的是,一般的对称性变换是容易发现的,无论是空间平移,旋转、还是CPT不变,都能自然的得到一些守恒量.
但有时并非那么直接就能得到对称性,这时就需要借助群论等方法进行研究发现相应的对称性；当然我们也可以猜守恒量,此时Noether定理可以起一个验证作用.
