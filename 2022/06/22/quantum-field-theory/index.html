作用量理论简介

这一节我们来回顾一下分析力学的基础概念.分析力学的整体框架是建立在哈密顿方程或者拉格朗日方程的体系上的,
而二者实际又相互联系.

对于一个粒子,我们利用广义坐标qiq{_i}qi​及其一阶导数q˙i\dot q{_i}q˙​i​描述它某一时刻的位置和运动趋势（速度）,利用Lagrange量描述它在某一时刻的状态,当粒子随时间变化开始运动,Lagrange量也随之变化.
而Lagrange量随时间的累计（即积分）相当于粒子运动的某种路径,反映了粒子随时间的演化情况,我们称它为作用量.
从状态a到b的作用量是:
$
S= \int_{ \rm{a} }^{ \rm{b} } {L\left( { {q_i},{ {\dot q} _i},t} \right)} {\rm d}t
$
（其中指标i标记变量坐标分量）

在分析力学中有最小作用量原理要求
作用量是处处极小的.即
δS≡0\delta S \equiv 0 δS≡0
利用变分法可导出熟知的Euler-Lagrange 方程:
ddt(∂L∂q˙i)−∂L∂qi=0 \frac{ {\rm d} }{ { {\rm d}t } }( \frac{ { \partial L } }{ { \partial { \dot q _i} } } ) - \frac{ { \partial L } }{ { \partial {q_i} } } = 0 dtd​(∂q˙​i​∂L​)−∂qi​∂L​=0

作用量可以看作L这个函数作为变量的函数,我们通常称这类函数为泛函,我们引入“泛函导数"的概念描述泛函的变化,对于一个泛函F[f(x)]F[f(x)]F[f(x)],它的泛函导数定义如下：
$
\frac{ {\delta {\rm{F} } } }{ {\delta f(x)} } = \mathop {\lim }\limits_{ \varepsilon \to 0 } \frac{ {F[f(x’) + \varepsilon \delta (x - x’)] - F[f(x’)]} } \varepsilon
$
